MAAS Coffee for Everyone

Empowering visually impaired individuals to independently order coffee

The Challenge

The MAAS Coffee for Everyone project aims to address a critical accessibility issue faced by visually impaired individuals when using touchscreen-equipped coffee machines. The primary problem is the lack of a user-friendly interface tailored to the needs of visually impaired users, making it difficult for them to independently operate these machines and enjoy the convenience of modern coffee-making technology.

In response to this challenge, the MAAS Coffee project sought to develop a mobile application that serves as a communication interface for MAAS touchscreen coffee machines. This application provides an intuitive and accessible platform, enabling visually impaired users to prepare coffee independently and conveniently by generating a QR-code, which can be scanned by a coffee machine from MAAS.
